Todo API

GET /todos

Fetches all todos from the database.

Doesn't need anything passed along.

GET /todos/:id

Fetches a specific todo from the database.

Needs the _.id and new todo object passed along.

POST /todos

Creates a new todo entry.

Needs the todo object passed along. 

                        
{
  "title": "",
  "categories": "",
  "content": "",
  "image": ""
}

DELETE /todos/:id

Deletes a specific todo.

Needs the _.id passed along.

PATCH /todos/:id

Edits an existing todo.

Needs the _.id and new todo object passed along. 

                        
{
  "title": "",
  "categories": "",
  "content": "",
  "image": "",
  "completed": "",
  "completedAt": ""
}
